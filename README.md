* * Project Overview
BloodConnect is a modern Full Stack Blood Donation Management Platform built with the MERN Stack.
এই project-এর মূল উদ্দেশ্য হচ্ছে blood donor এবং blood recipient-এর মধ্যে একটি secure ও efficient connection তৈরি করা।
The platform allows users to register as donors, create blood donation requests, search for available donors, manage donation activities through role-based dashboards, and contribute financially to support the organization.
The project focuses on clean UI, responsive design, authentication, authorization, secure APIs, and an excellent user experience.
* * Purpose
1. BloodConnect helps people find blood donors quickly and efficiently.
2. It provides a complete digital solution for:
3. Blood Donation Requests
